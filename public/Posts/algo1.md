# 자료구조 - 알고리즘

## 자료구조의 정의
- 데이터 처리를 위한 저장의 구조.
- 메모리를 효율적으로 사용하기.
- 문제 해결을 위해서도 자료구조 선택이 달라짐.
- Like : Stack , queue.
### Stack
- LIFO
- First Input First Out
> 후위연산자 계산기
### Queue
- FIFO 
- First Input First Out
>
